# 🎮 Tic Tac Toe Game

This is a simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 🚀 Features

* Two-player gameplay (X and O)
* Winner detection
* Reset button

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## ▶️ How to Play

1. Open the game in your browser
2. Players take turns clicking boxes
3. First to align 3 wins!
